title: Client API
---
 
NuAuth offers a client API that can be used to interact with the NuAuth server. The client API is available in the form of a method that can be used in the browser or in Node.js. The methods returned by the client API calls the NuAuth server to perform the requested operation such as login, logout, and refresh token.


<div className="steps [&_h3]:step">

### Configuration

```ts title="lib/client.ts"
import { createAuthClient } from 'NuAuth/client';
import type Config from '@/lib/config';

export const authClient = createAuthClient<Config>()
```
